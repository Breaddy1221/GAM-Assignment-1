## 📌 Project Overview
This project explores game production workflows using Unreal Engine and GitHub for version control.  
The aim is to develop small proof-of-concept features while understanding how different elements of game development integrate within a production pipeline.

---

## 🛠 Tools Used
- Unreal Engine  
- GitHub (Version Control)  

---

## 📋 Tasks Completed
- Built the environment using Unreal Engine landscape tools and imported assets  
- Adjusted mesh pivots to ensure accurate interaction and animation behaviour  
- Developed an interactive cabinet system using Actor Blueprint  
- Implemented collision detection with on-screen interaction prompt (“Press F”)  
- Created open and close door animation using Timeline and float track  
- Integrated sound effects to enhance interaction feedback  
- Designed Start Game UI in Adobe Illustrator and imported into Unreal Engine  
- Created UI using Widget Blueprint and displayed it on the player viewport  
- Implemented button interaction to start gameplay and remove UI  
- Developed a fire visual effect using Niagara to enhance the environment  

---

## 🧪 Interaction System
An interactive cabinet system was developed using Actor Blueprint. A collision box detects when the player is near the cabinet and displays a prompt to press “F” for interaction. Pressing “F” triggers a timeline animation to open the cabinet door, accompanied by a sound effect. Pressing “F” again reverses the animation to close the door, creating a simple and responsive interaction system. This demonstrates the use of event-driven logic and real-time interaction design in Unreal Engine.

---

## 🖥 UI Proof of Concept
A Start Game UI was designed and implemented using Unreal’s Widget Blueprint system. UI assets were created in Adobe Illustrator and imported into Unreal Engine. The interface appears on the player’s viewport at the start of the game, and clicking the “Start” button removes the UI and begins gameplay. This demonstrates basic UI integration, layout design, and user interaction flow.

---

## 🔥 Visual Effects
A fire effect was created using Unreal Engine’s Niagara system and applied to the stove area. This enhances environmental realism and demonstrates the use of real-time visual effects within the scene.

## 🔄 Version Control Workflow (GitHub)
GitHub was used to manage project files and track development progress in a solo workflow.

### Process:
- Created repository  
- Cloned project locally  
- Added and committed files  
- Pushed updates to GitHub  

This workflow allowed for version tracking, backup of project files, and clear documentation of development progress through commit history.
